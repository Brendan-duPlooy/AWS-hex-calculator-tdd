# Hexadecimal Calculator - Functional Requirements

## Input
- The system must accept hexadecimal number inputs (0-9, A-F)
- The system must restrict inputs to a maximum of 2 digits 
- The system must validate that input is a valid hexadecimal number 
- The system must prevent invalid characters from being entered 

## Artithmatic Operations
- The system must perform addition on two hexadecimal numbers 
- The system must perform subtraction on two hexadecimal numbers 
- The system must perform multiplication on two hexadecimal numbers 
- The system must perform division on two hexadecimal numbers 

## Output 
- The system must return results in hexadecimal format 
- The system must restrict outputs to a maximum of 4 digits 
- The system may not return negative results 
- The system may not return decimal values 

## Calculations
- The system must convert hexadecimal inputs into decimal for calculation 
- The system must convert results back into hexadecimal 
- The system must handle edge cases (example: division by zero) 

## User Interaction
- The system must allow users to input two numbers 
- The system must allow users to select an operation 
- The system must display the result clearly
